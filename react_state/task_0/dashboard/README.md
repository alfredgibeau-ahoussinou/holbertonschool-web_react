# Task 0 — Dashboard

Application Vite + Jest pour gérer l'état local des notifications (`displayDrawer`).

## Scripts

- Installer: `npm i`
- Développement: `npm run dev`
- Tests: `npm test -i`
- Lint: `npm run lint`
- Build: `npm run build`
- Preview: `npm run preview`

## Points clés
- `App` maintient `displayDrawer` et passe `handleDisplayDrawer/handleHideDrawer` à `Notifications`.
- Tests couvrent le clic sur "Your notifications" et le bouton de fermeture.
