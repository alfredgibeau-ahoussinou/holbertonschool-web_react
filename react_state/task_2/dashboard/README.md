# Task 2 — Dashboard

Contexte React: `App` gère `user`, expose `logIn`/`logOut` via Provider, `Login` consomme `logIn`.

## Scripts

- Installer: `npm i`
- Développement: `npm run dev`
- Tests: `npm test -i`
- Lint: `npm run lint`
- Build: `npm run build`
- Preview: `npm run preview`

## Points clés
- Provider autour de l'app avec `{ user, logOut }` depuis l'état local.
- Soumission valide rend la vue `CourseList`.
