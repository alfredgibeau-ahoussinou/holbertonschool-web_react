# Task 1 — Dashboard

`Login` en composant de classe contrôlé avec validation et bouton Submit activé selon critères.

## Scripts

- Installer: `npm i`
- Développement: `npm run dev`
- Tests: `npm test -i`
- Lint: `npm run lint`
- Build: `npm run build`
- Preview: `npm run preview`

## Points clés
- Champs contrôlés `email`/`password` + `enableSubmit`.
- Email valide (regex) et mot de passe ≥ 8 requis.
