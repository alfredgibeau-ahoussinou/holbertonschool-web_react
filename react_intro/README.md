## Getting Started

### Prerequisites
- Node.js 20.x.x or greater
- npm 10.x.x or greater

### Installation
```bash
# Navigate to the final task directory
cd task_5/dashboard

# Install dependencies
npm install

# Run tests
npm test

# Start development server
npm run dev

# Build for production
npm run build
```
